# Informacijski sustav za organizaciju festivala (Street Food Fest)

Ovaj projekt obuhvaća razvoj i implementaciju informacijskog sustava za digitalizaciju procesa planiranja i operativnog vođenja masovnih manifestacija, s primarnim fokusom na **Street Food Fest Široki Brijeg**.

## Cilj i Svrha
Glavni cilj projekta je povećanje učinkovitosti organizacije festivala kroz centralizaciju podataka o izlagačima, financijskim tokovima i ljudskim resursima. Sustav omogućuje:
* Napredno upravljanje bazom izlagača i logistikom štandova.
* Sustavno praćenje financija (prihodi, rashodi, sponzorstva).
* Analitiku povijesnih podataka za optimizaciju budućih izdanja festivala.

Projekt se izvodi prema **iterativnom modelu** ćivotnog ciklusa.
